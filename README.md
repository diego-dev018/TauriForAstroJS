# Tauri For AstroJS

Este es un ejemplo de como se puede integrar Tauri con AstroJS. Crea aplicaciones de escritorio con tecnologías web modernas.

## Requisitos

- NodeJS
- Rust
- Tauri CLI

## Instalación

```bash
git clone https://github.com/diego-dev018/TauriForAstroJS
```

```bash
cd TauriForAstroJS
```

```bash
npm install
```

## Ejecutar

```bash
npm run tauri:dev
```

## Compilar

```bash
npm run tauri:build
```
